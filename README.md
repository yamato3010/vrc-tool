# vrc-tool(仮名)

VRChatterのための支援ツールです。  
現在開発中。
 
# DEMO
 
 画像は開発途中のものです。

# Features
 
- フレンドのオンライン状況一覧表示
- フレンドの詳細情報表示（未実装）
- インスタンスの詳細情報表示（未実装）
- イベント管理機能（未実装）
- 所属グループ通知一覧表示（未実装）
 
# Note
 
- 本アプリケーションはVRChatAPIを使用しています。
- VRChatAPIは公式にサポートされていないため，予期せぬ仕様変更やAPIエンドポイントへのアクセス中断により，本アプリケーションが使用できなくなる可能性があります。
- 自己責任での使用をお願いいたします。
